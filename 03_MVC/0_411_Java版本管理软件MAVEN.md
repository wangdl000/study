## Java版本管理软件MAVEN  

### maven到底是什么  
Maven是基于项目对象模型(POM project object model)，可以通过一小段描述信息（配置）来管理项目的构建，报告和文档的软件项目管理工具。  
（Maven的核心功能便是合理叙述项目间的依赖关系）  
通过pom.xml文件的配置获取jar包，而不用手动去添加jar包。
